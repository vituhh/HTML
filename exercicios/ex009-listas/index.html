Trabalhando com listas

Listas ordenadas

Para criar uma lista ordenada, ou seja, uma lista onde a ordem importa, utiliza-se <ol>. E para cada elemento dessa lista utiliza-se <li>

Exemplo de código:

<ol>

<li>Elemento1<li>

<li>Elemento2<li>

<ol>

Exemplo de como fica:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Pode-se utilizar <ol type = ""> para mudar o tipo de contagem na frente

Listas não ordenadas

Para criar uma lista não ordenada, ou seja, uma lista onde a ordem não importa, utiliza-se <ul>. E para cada elemento dessa lista utiliza-se <li>

Exemplo de código:

<ul>

<li>Elemento1<li>

<li>Elemento2<li>

<ul>

Exemplo de como fica:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Pode-se utilizar <ul type = ""> para mudar o tipo de contagem na frente. Existem os seguintes type para ul: disc, circle e square

Aqui utiliza-se <dt> para o termo, e em seguida, utiliza-se <dd> para indicar a definição daquele termo

Exemplos:
