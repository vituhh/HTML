Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>(não semântica).

Nesta frase, tenho um termo em negrito usando a tag <strong>(semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>(não semântica)

Nesta frase, temos um termo em itálico usando a tag <em>(semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Nesta frase, temos um termo grande usando a tag <big>(não semântico e está em desuso)

E nesta frase, temos um termo pequeno usando a tag <small>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins>.

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub>
