Código-fonte

O comando document.getElementByID('teste') é escrito em linguagem JavaScript. A tag <code> é usada para que todas as letras ocupem espaços iguais, porém, ele não organiza os códigos de forma correta.

num = int(input('Digite um número:')) if num % 2 == 0: print('O número {num} é PAR') else: print('O número {num} é ÍMPAR')

Para organizar o código de forma mais bonita, utilizamos a tag <pre> 

        
    num = int(input('Digite um número:'))
    if num % 2 == 0:
        print('O número {num} é PAR')
    else:
        print('O número {num} é ÍMPAR')
    print('Fim do programa')
