Usando links

Usando links externos

Para você adicionar um link externo ao seu site, você utiliza a tag <a href="endereço">. Para ficar mais fácil, envelope a frase que servirá como âncora para o site externo e digite a.

Exemplo de código:

<p>Você pode acessar o meu <a href="https://gustavoguanabara.github.io">repositório público no GitHub</a><p>

Exemplos de como fica:

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTube

Forma melhor:

Para que, ao clicar na âncora, a aba do seu site não seja sobreposta, digite <a href="endereço" target="_blank" rel="external">. Com isso, abrirá outra aba.

Exemplo de código:

<p>Você pode acessar o meu <a href="https://gustavoguanabara.github.io" target="_blank" rel="externa">repositório público no GitHub</a><p>

Exemplos de como fica:

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTube

Usando links internos

Para que você crie uma âncora para uma página interna do seu site, é necessário que a página que será acessada esteja na mesma pasta da primeira.

O código fica da seguinte maneira na primeira página:

<p>Esta é a primeira página do site. Se você quiser, pode acessar também a minha <a href="pag002.html" rel="next">segunda página</a><p>

O rel="next" indica que está indo para uma página seguinte

Enquanto na segunda página o código fica assim:

<a href="index.html" rel="prev">Voltar para a primeira página</a>

O rel="prev" indica voltar para página anterior

Como fica na prática:

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página

Você também pode acessar a nossa página de notícias
