Imagem dinâmica

Para fazer com que uma imagem seja dinânima, ou seja, altere o tamanho de acordo coma tela, utiliza-se a tag <picture>. E dentro dela, utiliza-se a tag <source>. Escolha a 'source:media:type'.

Exemplo de código:

<picture>

<source media="(max-width: 750px)" srcset="pasta_com_imagens/nome_imagem_menor.png" type="image/png">

<source media="(max-width: 1050px)" srcset="pasta_com_imagens/nome_imagem_media.png" type="image/png">

<img src="pasta_com_imagens/nome_imagem_grande.png" alt="Imagem flexível">

</picture>

Observação importante:

  • É importante que esta seja a ordem do código:
    1. source para imagem menor
    2. source para imagem media
    3. img para imagem maior

Funcionamento na prática:

Tente abrir esse site em vários dispositivos diferentes, ou simplesmente aumente e diminua o tamanho do seu navegador

Áudios

Primeira forma

Para reproduzir um áudio, utiliza-se a tag <audio>

Exemplo de código:

<audio src="pasta_com_audio/nome_audio.mp3" controls autoplay></audio>

Observações:

  • Este formato de código só funciona para áudios nos seguintes formatos:
    • MP3
    • WAV
    • OGG
  • Os parâmetros 'controls' e 'autoplay' devem ser adicionados
  • O parâmetro 'controls' faz com que apareça o botão de play
  • O parâmetro 'autoplay' faz com que o áudio toque sozinho(nem sempre funciona)

Reproduzindo áudio

Segunda forma

Para reproduzir um áudio pode-se também utilizar a seguinte forma:

Exemplo de código:

<audio preload="metadata" controls loop>

<source src="pasta_com_audio/nome_audio.mp3" type="audio/mpeg">

<source src="pasta_com_audio/nome_audio.ogg" type="audio/ogg">

<source src="pasta_com_audio/nome_audio.formato" type="audio/formato_do_audio">

</audio>

Observações:

  • O programa sempre irá ler de cima para baixo
  • O formato MP3 é o único com nome diferente em relação ao type. No caso, fica type="audio/mpeg"
  • Parâmetros adicionados:
    • preload - Exstem três opções:
      • auto - faz o dowload de todas as informações do áudio assim que o site é iniciado e acaba deixando o site ainda mais pesado
      • metadata - faz o dowload de algumas informações do áudio quando o site é iniciado
      • none - não faz o dowload das informações do áudio quando o site é iniciado
    • controls - cria o botão para iniciar o áudio
    • loop - faz o áudio iniciar novamente quando acabar de ser tocado

Vídeos

Vídeos internos

Para colocar no site um vídeo que já está hospedado localmente, utiliza-se a tag <video>. Caso esteja fazendo no VS Code, apenas digite 'video' e aparecerá assim: <video src=""></video>

Exemplo de código:

<video width="tamanho_desejado" poster="imagem.jpeg" controls autoplay loop>

<source src="local_do_video.mp4" type="video/mp4">

<source src="local_do_video.mkv" type="video/mkv">

<source src="local_do_video.webm" type="video/webm">

<p>Seu navegador não tem compatibilidade com reprodução de vídeos</p>

</video>

Observações:

  • Os parâmetros 'width', 'controls', 'poster', 'autoplay' e 'loop' são adicionados manualmente
  • O parâmetro 'width' serve para redimensionar o vídeo no site
  • O parâmetro 'controls' serve para que seja possível dar play no vídeo
  • O parâmetro 'poster' serve para criar uma capa para seu vídeo utilizando uma imagem na mesma pasta do programa
  • O parâmetro 'autoplay' serve para que o vídeo inicie quando o site for iniciado
  • O parâmetro 'loop' serve para que o vídeo rode em loop
  • É importante colocar vários formatos de vídeo no código, pois dependendo da máquina, certos tipos de vídeos não irão rodar

Vídeos externos do YouTube

Utilizar vídeos externos vão dar menos custo para seu cliente.

Como fazer:

Para fazer isso, procure um vídeo no YouTube, clique em 'Compartilhar' e clique em 'Incorporar'. Com isso, o YouTube irá gerar um código. Copie e cole no seu programa o código.
